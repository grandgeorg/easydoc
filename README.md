## Install

Download [latest release](https://git.grandgeorg.de/Viktor/easydoc/releases) and unpack it.

```bash
cd ./easydoc/
npm install
# start vscode if you like ...
code .
```

## Usage

```bash
# watches on file changes and runs build:
npm run watch
# or build one time
npm run build
```
