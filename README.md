## Install

```bash
# clone via https:
git clone https://git.grandgeorg.de/Viktor/easydoc.git
# or clone via SSH (if you have a key):
git clone git@git.grandgeorg.de:Viktor/easydoc.git
cd ./easydoc/
npm install
# start vscode if you like ...
code .
```

## Usage

```bash
# watches on file changes and runs build:
npm run watch
# or build one time
npm run build
```
